# Implementation Plan - 2026-05-05

- [x] **Function Calls in Expressions** - DONE

  - Custom functions via Db.functions
  - Type inference working

- [x] **ILIKE Operator** - DONE

- [ ] **Flat Expression Parser**
  - Implement Pratt Parser or Shunting Yard Algorithm

  - Reduce nesting depth

- [ ] **O(1) AST Resolution**
